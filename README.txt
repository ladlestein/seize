= seize

* https://github.com/TheRealReal/seize

== DESCRIPTION:

Seize is a framework for importing data.

== FEATURES/PROBLEMS:

* Metadata-driven import
* Leverages ActiveRecord objects

== SYNOPSIS:

  rails generate importer <name>

and generate the RDoc.
